Touch Gesture
 - 작은 모바일 화면을 더 넓게 활용할 수 있음
 - 터치(Touch)
1. 탭(Tap)
>한 손가락으로 한 번 터치한 후 즉시 들어 올리는 것
보통 앱을 실행하거나 어떤 것을 선택할 때 사용
2. 더블 탭(Double Tap)
>한 손가락을 2회 연속 두드리는 것
지도나 이미지를 화면에서 확대하거나 축소하기 위한 제스처로 사용
3. 프레스(Press)
>화면 위를 일정 시간 동안 약간의 힘을 가한 상태로 탭 하는 제스처
숨겨진 메뉴를 불러오거나 관련 기능을 실행할 때 사용
4. 팬(Pan)
>손가락을 떼지 않고 계속 드래그(Drag) 하는 제스처
오브젝트를 이동할 때, 라인 그리기 혹은 확대된 이미지를 상하좌우로 움직여 탐색할
때 사용
5. 스와이프(Swipe)
 - 좌우
>손가락을 댄 후, 일직선으로 드래그하는 것
보통 화면 탐색 시, 스크롤(Scroll) 기능으로 사용
6. 플릭(Flick)
 - 스크롤
>스와이프에서 좀 더 빠르게 한 방향으로 미는 것
화면을 빠르게 스크롤 하거나 회전할 때, 오브젝트를 무작위 방향으로 던질 때
*스와이프
 - 좌우 이동하는 포스팅
**그리드
 - 타일처럼 포스팅
7. 드래그 (Drag)
>오브젝트를 이동할 때나 정해진 방향으로 움직인 후 손가락을 떼는 행동
오브젝트를 이동시킬 때나 텍스트를 복사할 때 주로 사용
8. 핀치 인/아웃 (Pinch in/out) 
>손가락 두 개를 화면에 댄 상태에서 두 손가락 사이를 넓히거나(out) 좁히는(in) 제스처
지도나 이미지를 볼 때 확대/축소 기능
9. 로테이트 (Rotate) 
>두 손가락을 이용하여 오브젝트가 도는 회전하게 사용하기 위한 제스처
보통 화면에서 이미지를 회전시킬 때 사용되는 제스처

화면 디자인 UI & button
1. 도시락 Menu Icon (bento)
>여러가지 메뉴 icon이 박스 형태로 모여 있음을 표현
 - 예) 구글 ui
2. Hamburger Menu Icon
>하위 목록이 존재한다는 표현
 - 줄3개
3. Meatball Menu Icon
>더 많은 메뉴의 옵션이 있다는 icon
 - 점 3개 가로
4. Kebab Menu Icon
>더 작은 하위 메뉴를 열 때
추가 옵션이 있는 메뉴가 있다는 icon
 - 점 3개 세로
 - 미트볼보다 영역 적게 차지함, 요즘 많이 사용하는 추세
5. Placeholder Text
 - 검색어 입력창 등에서, 도움을 주기 위해서 미리 입력되어 있는 텍스트
 - 입력하면 사라짐
*LP(랜딩페이지) : 페이지 이동
6. CheckBox / Radio / Toggle ★
Check box
 - 중복 가능
 - 체크박스는 말그대로 네모난 상자를 클릭하면 체크 표시가 표출되는 UI
 - 체크 박스는 확인, 동의 했다는 의미일 때
 - 중복으로 값을 선택할 수 있을 때
 - 값이 없어도 될 때 (선택 사항)
Radio box
 - 중복 불가
 - 라디오 버튼은 말그대로 동그란 버튼으로 표출되는 UI
 - 옵션 중 하나 선택
 - 값이 없는 상태가 존재하면 안될 때
 - 선택지 중 하나만 골라야 할 때
Toggle box
 - on/off ui단순화
 - 즉각적으로 변화가 눈에 보일 때
 - 두 가지 중 하나의 상태를 설정할 때
7. Floating Action Button
 - 모페이지(기존 페이지)
 - 뭔가를 띄움
 - 마케팅 or 메뉴
 - 사용자가 특정 작업을 수해할 수 있도록 행동을 유도하는 button
 - 원형의 형태로 스크린 내 가장 앞에 활성화 노출
8. PopUp(Layer PopUp)
 - 필수 정보 공유
> 중요한 정보를 전달할 때, 사용자의 의사결정이 필요한 경우
> 반드시 확인이 필요한 정보를 전달 할 경우 노출
> 광고와 관련된 알림을 보여줄 경우 노출
 - 시스템 팝업(pc혹은 단말에서 띄움), 레이어 팝업(클라이언트가 앱에서 띄움)
9. Bottom sheet
 - 쇼핑몰
 - 기존 화면과 연관성이 있는 콘텐츠를 제공하거나 화면의 이동없이 다양한 메뉴를 이용해야 할 때 사용
10. Splash Screen(APP 구동)
 - 기존 화면과 연관성이 있는 콘텐츠를 제공하거나 화면의 이동없이 다양한 메뉴를 이용해야할 때 사용
11. GNB(Global Navigation Bar)
(기획자마다 상이하게 진행할 수도 있음)
 - 최상단의 공통적인 목적을 가진 영역
 - 하나의 화면에 거의 모든 page를 한곳에 Menu 형식으로 모아 둔 영역
 - 대부분 page 상단에 노출
 - SNB(Side Navigation Bar) / BNB(Bottom Navigation Bar)
12. SNB(Side Navigation Bar)
(기획자마다 상이하게 진행할 수도 있음)
 - 옆에
 - LNB(왼쪽 SNB), RNB(오른쪽 SNB)
 - 보편적으로 화면 상단이 아닌 좌/우측에 Menu 형식으로 모아둔 영역
13. FNB(Foot Navigation Bar)
(기획자마다 상이하게 진행할 수도 있음)
 - 화면 하단에 copyright / 연관 site / 보조 메뉴
14. ToolTip
 - 보통 info button을 클릭할 경우 관련 정보를 간단하게 레이어 형식으로 노출되며 text와 함께 노출
